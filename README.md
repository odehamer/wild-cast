## Set-up

Create a virtual environment for storing your project dependencies:

```
python3 -m venv .venv
```

Activate the virtual environment:

```
source .venv/bin/activate
```

Install project dependencies:

```
pip install -r requirements.txt
```

## Execution

Run the project

```
python main.py
```
